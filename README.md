# Spots On Map API

Spots On Map is a node.js API to work with the Spots on Map Application

## Installation

Use the package manager [npm] to install dependencies.

```bash
npm install
```

## Start the server

```bash
npm start
```

## Build and run productions files

```bash
npm build
npm prod
```
